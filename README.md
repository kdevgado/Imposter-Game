# DDD Game Hub

DDD Game Hub is a collection of pass-the-phone party games plus live online UNO and trivia rooms.

## Download and install

You do not need to install developer tools or run any code. Choose your device and open the downloaded installer.

### [Download for Android](https://github.com/kdevgado/ddd-gamehub/releases/latest/download/DDD-Game-Hub-Android.apk)

Download the APK, open it, and tap **Install**. Android may ask you to allow installation from your browser or Files app the first time.

### [Download for Windows](https://github.com/kdevgado/ddd-gamehub/releases/latest/download/DDD-Game-Hub-Windows-Setup.exe)

Download and open the installer, then follow the setup window. Because the installer is not currently code-signed, Windows SmartScreen may require **More info** → **Run anyway**.

You can also browse [all available releases](https://github.com/kdevgado/ddd-gamehub/releases/latest).

## Notes

- Pass-the-phone games work offline after installation.
- Live UNO and trivia rooms require an internet connection.
- The Android download is debug-signed for direct installation and sharing. A Google Play release requires a private upload key.
- If the repository is private, GitHub will ask you to sign in before downloading.
